# 工作流：pass2-bookify（Pass 2 书面化 · 发表级编辑）

> ⚠ **前置硬条件**：只能在 **Pass 1 整理稿已经受访人审阅签字** 之后运行。未签字不得进入 Pass 2。这是口述史伦理底线。
>
> 书面化方法论由合作者 **Kelvin（中科大 科技史）** 从张藜 / 中华规范 / 陈墨 / 熊卫民 / 《中国科技史杂志》等系统梳理；框架概念归属各原始作者。

## Pass 1 vs Pass 2 的根本区别

| | Pass 1 去口语化（主流程） | Pass 2 书面化（本工作流） |
|---|---|---|
| 输入 | 录音转写稿 | **已签字的** Pass 1 整理稿 |
| 目标 | 忠实、可读、可签字（≈档案本/初整稿） | 发表级：流畅、有结构、合学术体例（≈传播本/整理稿） |
| 篇章重排 | ❌ 仅文末提示时间线 | ✅ 可按主题/时序重组 |
| 提炼删节 | ❌ 不删实质 | ✅ 可删冗余、合并重复主题（保留关键史实与原意） |
| 口语质感 | 大量保留 | 转书面语体，但**不抹个性** |
| 不变底线 | 不捏造 | **仍不捏造**；引文/史实/原意/立场不可改 |

## 流程（与 Pass 1 复用同一项目，新开 `pass=2`）

1. 复制签字稿：`output/整理稿.md` → `output/书面稿_draft.md`，manifest 标 `pass: 2`。
2. 读 `references/methodology.md` §3（双本制/生熟）明确"现在允许重组、但仍不可捏造"。
3. 按下列 §1–§4 逐节书面化，**记录重排映射**（书面稿每节 ← 来自签字稿哪几段）。
4. 史实守门：人名/机构/专名沿用 `term_lock.md`；引文逐字核对签字稿。
5. 产出 `output/书面稿.md` + `review/Pass2重排映射.md`；发表前仍建议受访人/版权方复核。

## §1 句式操作

**短句合并为复句**（口语短句堆砌 → 有逻辑关系的书面复句）：
- 因果→因此/故而 · 递进→而且/进而 · 转折→然而/尽管如此 · 并列→同时/此外
- 例：「那时候条件很差。设备也没有。经费也不够。」→「当时条件很差，既无设备，经费亦不充足。」

**语序调整**：口语倒装恢复（"去了昆明我们就"→"我们就去了昆明"）；插入成分整理（"张先生他呢，就是我们所长"→"张先生即我们所长"）。**但保留有修辞效果的倒装**（"这些事情啊，我是想不通的"可留）。

**补全省略**：补主语/宾语；模糊指称（他/那个人/当时的领导）→ 替换为具体姓名或身份。**但不补到啰嗦**——一个"他"能说清就不硬加名字。

**控制句长**：主体陈述句 15–35 字；复杂叙述不超 50 字；极端短句仅在有修辞目的时保留。

## §2 结构重组

- **按话题重新分段**：散落各处的同话题问答合并；每段 100–400 字；段首不出现"还有/另外/对了"。
- **添加小标题**：按话题或时段分节（如"一、入职中央研究院的经过"）。
- **时间线锚定**："后来/那时候"→ 尽量补出具体年份；不留指代不明。
- **叙事节奏**：关键场景细节还原；过渡内容一笔带过；纯跑题可删并标注"此处略去与主题无关的对话"。

## §3 用词规范

**适度替换**（仅在正式叙述中）：搞研究→从事研究 · 那会儿→当时/彼时 · 挺不错→颇为可观 · 说白了→简言之。
**不替换**：受访者**关键判断用词**不软化（"完全是错误的"不改成"值得商榷"）；历史特定称谓保留并视情加注；直接引语原样；**受访者标志性措辞**（如"笑死人了"）保留。

## §4 学术标注 — 脚注五功能体系（熊卫民）

| 功能 | 场景 | 示例 |
|---|---|---|
| 人物注 | 重要人物首次出现 | 「竺可桢（1890–1974），气象学家，时任中央研究院气象研究所所长。」 |
| 事实核查注 | 受访人记忆有误 | 「口述者此处记忆不确，该事件实发生于 1947 年。」 |
| 文献出处注 | 提及出版物/档案 | 「参见《中央研究院院务月报》1943 年第 5 期。」 |
| 交叉印证注 | 不同受访人说法矛盾 | 「据笔者对邹先生的访谈，其讲法为……与此处有出入。」 |
| 补充说明注 | 补背景 | 「1959–1961 年间因自然灾害，研究所伙食一度极困难。」 |

随文注：整理者补充用（ ）；受访人审阅添加用 ［ ］；编者注署名【编者注：…】。篇末标注：是否经审阅 / 初发信息 / 修订情况。

## §5 书面稿风格与体例

**三种叙述声音分工**：导言（整理人旁白，学术书面语，第三人称，不煽情）/ 访谈人提问（精炼、引导性）/ 受访人回答（口语化的书面语，有叙事节奏，保个性）。

**叙事特征**：个人经历+时代背景双线（背景自然植入不说教）；科研过程保具体细节；关键瞬间特写、过渡简洁；情感由事实自然流露、不用感叹号、正文克制；多声部并存、整理人不当裁判。

**完整体例**：标题（含受访人）→ 受访人简介(200–500字) → 导言(150–350字：缘起+问题意识+史学价值) → 时间地点 → 正文(小标题分节, Q&A) → 文末注记 → 脚注。

**按受访人类型调策略**：科学家口述（Q&A 最纯粹、回答较长、保个性）/ 管理干部（更结构化、时间线更清晰）/ 知识型回顾（可转自述体、脚注更详）。

## §6 绝对禁止

- ❌ **唐德刚式改写**：读者分不清哪句来自受访者、哪句来自整理者。
- ❌ **去采访人化**：删掉采访人提问只留陈述。
- ❌ 编造任何事实/引文/数据；选择性引用制造误导。
- ❌ 把口语完全改成公文/论文腔；用"书面化"当消毒手段抹掉个人语言痕迹。
- ❌ 替受访人裁决其保留的矛盾/不确定（可脚注说明，但不擅自定真伪）。

## §7 Stage 2 自检

句式合并未改逻辑关系 · 关键判断/评价未被软化 · 脚注不越界（不确定标"疑"）· 编者注与原话区分清晰 · 未添原文没有的事实 · 提问仍在（未去采访人化）· 分段合理 · 时间线尽量补出具体年份 · 受访者金句与口语节奏仍有体现。
